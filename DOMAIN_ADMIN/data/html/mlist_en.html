There's 3 types of mail address list
  1. ordinary address list

    2. ordinary address list need to be edited manually, member can be any email address either inside domain or outside domain. please click "expand to edit" for insert or delete member after the address list is created
  2. department address list

    3. department address list includes all addresses of that department, it's not necessary to edit the member manually. the mail address is the department name
  3. whole domain address list

    4. the whole domain address list includes all addresses of the domain, it's not necessary to edit the member manually. the mail address can be any in domain
privilege is needed for expanding the mail address list, there're 3 types privilege for expanding mail address list
  1. free to expand

    2. anyone can send email to the address list
  2. internal members

    3. only internal member can send email to the address list, or email will be bounced
  3. inter-domain

    4. only address of same domain name can send email to the address list, or email will be bounced
  4. specified senders

    5. only specified senders can send email to the address list, or email will be bounced
caution! number of ordinary address list is 20, each ordinary address list can include maximum 200 mail addresses
