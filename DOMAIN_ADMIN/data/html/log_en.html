The "Search Logs" page contains
  1. ip address

    2. You can input ip address of sender's mail system, for example: 202.108.9.10, if you leave this blanket, all ip addresses will be included in search result
  2. rmote address

    3. You can input sender or recipients' address, for example: test@mail.com, if you leave this input blanket, all addresses will be included in search result
  3. mail type

    4. there're
    1. not selected --- all

      2. this is default option, if it is choosed, all types will be inclued in search result
    2. normal mail

      3. mail delivered to mail system
    3. reject mail

      4. the gateway find some spam information or some limited content in the mail, and it will be rejected by gateway, the original sender will get a bounce message in case of false judement
    4. virus mail

      5. mail contains virus, also be refused by gateway
    5. insultaed mail

      6. accepted by gateway but not relayed to mail system, normally, the sender will get a bounce message, which indictae the sender to activate the insulted message, if the sender follows the indication, the message will be activated and relayed to mail system
    6. nouser mail

      7. when gateway relay mail to mail system, it gets "user not exist" message from mail system, normally sender will get bounce message
    7. timeout mail

      8. remote delivery system fails to send mail to gateway because of internet communication problem
    8. retrying mail

      9. because of gateway's capability of processing email, on certain time, the gateway cannot process more mail of limimtation, under this condition, remote delivery will retry serveral times to make the recipient(s) get the mail
    9. out-going mail

      10. mail delivered by mail system

  4. start time

    5. search the log result from start time, blanket means all time of log saved in gateway
  5. end time

    6. search the log result before the end time(include the end time), blanket means before the last day
